# Employee Analysis using Apache Spark

This project demonstrates employee data analysis using Apache Spark (PySpark). It covers essential data processing operations such as loading datasets, filtering records, grouping data, calculating salary statistics, sorting information, creating new columns, and handling missing values.

The goal of this project is to understand how PySpark processes large datasets efficiently while applying common data analysis and preprocessing techniques.

## Features
- Dataset loading and schema inspection
- Data filtering and selection
- Department-wise grouping and aggregation
- Salary statistics calculation
- Data sorting and ranking
- Adding new columns
- Handling null/missing values
- Practical implementation with examples

## Technologies Used
- Apache Spark
- PySpark
- Python

## Learning Outcomes
This project helped strengthen understanding of:
- Big Data processing concepts
- PySpark DataFrame operations
- Data preprocessing techniques
- Aggregation and transformation methods
- Efficient handling of large datasets

## Project Purpose
The project was created as a practical learning exercise to explore employee data analysis workflows using Apache Spark.
